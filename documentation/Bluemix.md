# Bluemix

## Signup

- Go to [IBM Bluemix Console](https://console.ng.bluemix.net/)
- Sign Up
- Validate Email Address
- Welcome to [IBM Bluemix Dashboard](https://console.ng.bluemix.net/?direct=classic)!

### Dashboard Components
    
- Cloud Foundry Apps
- Containers
- Virtual Machines
- Data & Analytics
- Services & APIs
